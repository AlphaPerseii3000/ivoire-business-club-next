## Leçons Clés

1. **Les composants tiers décoratifs (React Bits) sont un multiplicateur de bugs** — Chaque composant a généré son propre problème. Stabiliser dans un test dédié avant intégration, ou réduire l'ambition visuelle.

2. **L'audit UX post-DS fonctionne** — Impact massif pour un delta de code minimal. À systématiser pour toute story landing/page publique.

3. **Les Dev Notes ne contrôlent pas les DS subagents** — L'instruction "npm install avant build" doit être explicite dans chaque DS dispatch context, pas juste documentée.

4. **Le pattern vidéo scroll-driven (playback-rate + rAF rewind) est un asset réutilisable** — Documenté, testé, applicable à d'autres projets.
